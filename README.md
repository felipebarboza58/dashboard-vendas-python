#  Dashboard de Vendas

Projeto de análise de dados desenvolvido com Python e Streamlit.

##  Funcionalidades
- Filtros por categoria, região e período
- KPIs de vendas e lucro
- Gráficos interativos
- Insights automáticos

##  Tecnologias
- Python
- Pandas
- Streamlit
- Plotly
